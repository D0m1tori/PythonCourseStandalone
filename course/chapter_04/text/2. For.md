### Цикл `for` в Python с `range`

Цикл `for` в Python используется для повторного выполнения блока кода несколько раз. Если вы уже знакомы с циклом `while`, то цикл `for` можно рассматривать как удобный способ организовать итерации, когда заранее известно, сколько раз нужно выполнить код. В этом разделе мы рассмотрим цикл `for` с генератором чисел `range`, не затрагивая сложные структуры данных.

#### Синтаксис цикла `for`

Цикл `for` имеет следующий базовый синтаксис:

```python
for переменная in последовательность:
    блок_кода
```

- **`переменная`** — это переменная, которая на каждой итерации цикла будет принимать новое значение.
- **`последовательность`** — это объект, который предоставляет набор значений для каждой итерации цикла. В нашем случае это генератор чисел `range`.
- **`блок_кода`** — это код, который выполняется на каждой итерации.

#### Генератор `range`

Генератор `range` создает последовательность чисел. Мы можем использовать его в цикле `for`, чтобы задать, сколько раз и для каких чисел нужно выполнить код.

#### Пример 1: Использование `for` с `range(n)`

Если нужно выполнить цикл 5 раз, можно использовать `range(5)`. В этом случае переменная будет принимать значения от 0 до 4.

```python
for i in range(5):
    print("Итерация", i)
```

**Вывод:**

```terminal
Итерация 0
Итерация 1
Итерация 2
Итерация 3
Итерация 4
```

Здесь цикл `for` выполняется 5 раз, и на каждой итерации переменная `i` принимает очередное значение из диапазона чисел от 0 до 4.

#### Пример 2: Использование `for` с `range(start, stop)`

Если нужно начать с другого числа, например, с 3, и закончить на 7, используем `range(3, 8)`:

```python
for i in range(3, 8):
    print("Итерация", i)
```

**Вывод:**

```terminal
Итерация 3
Итерация 4
Итерация 5
Итерация 6
Итерация 7
```

Здесь цикл выполняется 5 раз, начиная с числа 3 и заканчивая числом 7.

#### Пример 3: Использование `for` с `range(start, stop, step)`

Если нужно выполнить цикл с шагом, отличным от 1, например, с шагом 2, можно использовать `range(start, stop, step)`. Пример для чисел от 0 до 9 с шагом 2:

```python
for i in range(0, 10, 2):
    print("Итерация", i)
```

**Вывод:**

```terminal
Итерация 0
Итерация 2
Итерация 4
Итерация 6
Итерация 8
```

Здесь цикл выполняется для значений 0, 2, 4, 6 и 8.

#### Использование цикла `for` без явного счетчика

Цикл `for` не всегда используется для работы с числами. Иногда можно просто выполнить блок кода заданное количество раз, не используя переменную-счетчик. В этом случае `range(n)` будет полезен, и переменную цикла можно игнорировать, если она не используется:

```python
for _ in range(3):
    print("Выполнение блока кода")
```

**Вывод:**

```terminal
Выполнение блока кода
Выполнение блока кода
Выполнение блока кода
```

Здесь переменная `_` используется как индикатор, что переменная не важна.
